⚙ 模型说明

第一步：ELO 评分 → 胜率

P(A 胜) = 1 / (1 + 10(ELOB - ELOA) / 400)

第二步：胜率 → 预期进球数

200 ELO 分差 ≈ 1 球优势；世界杯场均进球约 2.6 球

第三步：泊松分布模拟比分

P(k) = λk · e-λ / k! — 双方独立采样

第四步：蒙特卡洛模拟

10,000 次完整锦标赛 = 每次 72 场小组赛 + 32 场淘汰赛

⚠ 免责声明：本预测仅基于公开 ELO 评分和历史统计模型，不构成任何博彩或投资建议。 足球比赛结果受伤病、红黄牌、天气、临场状态等大量不可预测因素影响。 约 11 支低排名球队的 ELO 评分基于 FIFA 排名线性回归估算（标记为"估算"）。 数据来源：worldfootballrankings.com / FIFA / roadtowc.com / OneFootball。
